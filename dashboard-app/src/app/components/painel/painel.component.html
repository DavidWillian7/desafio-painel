Painel de Dados COVID-19

Introdução ao Dataset

Bem-vindo ao nosso painel de dados sobre a COVID-19. Este painel utiliza um conjunto de dados abrangente que oferece informações valiosas sobre a pandemia global.

Os dados são coletados do DataSet do Kaggle, sendo posteriormente armazenado e enviado pela API. O conjunto inclui:

  • Dados de vacinação em nível de país.
  • Informações sobre fontes de vacinação.

Este DataSet permite uma visão detalhada e atualizada do progresso da vacinação e outros aspectos relevantes da pandemia em escala global.
